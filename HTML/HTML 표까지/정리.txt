ol = 오덜 리스트(순서가 있는 리스트)
ul = 언오덜 리스트(순서가 없는 리스트)

li = 리스트 아이템(값을 입력할 수 있음)

--------------------------------

table = 표를 만들 때 사용되는 속성으로 td의 상위요소

tr = 행을 설정해줌

td = 열을 설정해줌

(행을 설정 후 열 설정)

--------------------------------
input

checkbox = 사용자가 체크 가능한 체크박스 생성

text = 사용자가 입력가능한 텍스트 생성

radio = checkbox와 같이 사용자가 체크 가능한 박스를 생성하지만 값이 여러개여도 하나의 속성이지 않는 이상은 하나만 선택가능

name = radio에 꼭 따라다니는 속성으로 태그를 설정해 일종의 묶음을 만든다(radio에서만 사용)

checked = checkbox 또는 radio에 이 값을 입력하면 입력한 요소에 사용자가 체크를 하지않아도 체크가 되있는다.(변경가능)
 
label = 글자 요소를 클릭해도 체크가 되게 만들어준다. (radio와 checkbox의 부모요소)

placeholder = 사용자에게 어떠한 값을 넣어야하는지 힌트를 줄 수 있다.

disabled = text 전용으로 상자안에 문자를 입력못하게 만든다.

--------------------------------

글자,상자 : 요소가 화면에 출력되는 어떠한 특성

인라인 = 글자를 다루는 요소들

블록 = 상자를 다루는 요소들
