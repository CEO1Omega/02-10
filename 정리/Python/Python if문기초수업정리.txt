if = 참을 나타내는 변수(항상 주인공이 왼쪽) / 여러개라도 실행
else = 거짓을 나타내는 변수 / if가 실행 성공하면 실행X
elif = 만약을 나타내는 변수 / if가 실행 성공하면 실행X

-----------------------------------------------------------------------------------------------------------------------
or = 하나라도 참이라면 된다(또는)
and = 여러개가 참이라도 된다.(그리고)
in = 안에 특정값이 있는지 확인(있나)
